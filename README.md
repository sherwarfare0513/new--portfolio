# Portfolio Website

A modern portfolio website featuring animations and responsive design.

## Features

- Home section with fade-in animation
- About section
- Projects section with hover effects
- Contact form
- Smooth scrolling and intersection observer animations
- Hire form with Email or WhatsApp choice

## Hire Form Behavior

- `Email Direct` sends the hire request using `formsubmit.co` to `sherwarfare0513@gmail.com`
- On the first live email submission, FormSubmit may ask you to confirm or activate the form through email
- `WhatsApp` opens a prefilled WhatsApp chat because normal static websites cannot auto-send WhatsApp messages without the WhatsApp Business API and a backend

## How to Run

Open `index.html` in a web browser.
